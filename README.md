# Claude Skills Share

A community repository of shareable Claude skills. Each folder contains a self-contained skill with all supporting files needed to load it into Claude.

## How to Use a Skill

1. Browse to the skill folder you want
2. Download the files inside (click a file, then click **Raw** or the download icon)
3. Upload the files to Claude when starting a conversation

To add a new skill, create a new folder with a descriptive name and include all necessary files. See an existing skill folder for reference on structure.
